Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript.The easiest way to access a single element in the DOM is by its unique ID. 1. `getElementById("id")` – Selects an element by its ID.
2. `getElementsByClassName("class")` – Selects elements by class name (returns a collection).
3. `getElementsByTagName("tag")` – Selects elements by tag name (collection).
4. `querySelector("selector")` – Selects the first matching element using CSS selectors.
5. `querySelectorAll("selector")` – Selects all matching elements (NodeList).br

Question-2: What is the difference between innerHTML, innerText, and textContent ?

TextContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.Difference between `innerHTML`, `innerText`, and `textContent`:
1. `innerHTML` – Gets or sets HTML content, including tags.
2. `innerText` – Gets or sets only visible text, ignoring hidden elements.
3. `textContent`– Gets or sets all text, including hidden content, without parsing HTML.
